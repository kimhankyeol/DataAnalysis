1.머신러닝이란
인공지능 연구과제중의 하나로 인간의 뇌가 자연스럽게 수행하는 학습이라는 능력을 컴퓨터로 구현하는 방법임

자세히 설명하면 
어느 정도 규모가 있는 샘플 데이터를 입력으로 넣어 분석시킴
그리고 이때 데이터에서 일정한 규칙을 찾아냄 
찾아낸 규칙을 기반으로 다른 데이터를 분류하거나 미래를 예측하는것

예를 들어 손글씨 문자 인식을 생각해봄
일단 수많은 손글씨 문자 데이터를 입력하고 이러한 데이터를 기반으로 일정한 규칙이 있는지 찾아냄
규칙을 찾으면 새로운 손글씨 문자를 입력했을때 해당 문자가 어떤 문자인지 예측할 수 있게됨
최근에는 머신러닝을 활용해 문자인식, 음성인식, 바둑 또는 장기 등의 게임전략, 의료진단 , 로봇 개발 등이 이뤄지고 있음

2. 어떻게 특징과 규칙을 찾을까 
머신러닝은 간단히 얘기하면 계산을 통해 구분선을 찾아내는것

구분선을 찾을떄 점과 점사이의 거리를 구함 
그리고 이러한 거리가 가까우면 비슷한 데이터 라고 판점함

3. 특징 추출

chapter 1,2,3 머신러닝을 목정으로 다양한 데이터를 모으는 방법을 알아봄

머신러닝을 하려면 데이터가 어떤 특징을 가지고 있는지 찾고 벡터로 만들어야 합니다. 이같은 변환처리를 특징추출이라고함

예를 들어 메일을 학습기에 입력한다고 하면 이떄 해당 텍스트에서 어떤 특징을 사용할 것인지 찾고 벡터로 변환해야함
예 ) 글자의 출현빈도,단어의 출현 빈도 를 입력해야됨

일반적으로 사용되는 머신러닝 프레임워크는 범용적인 용도의 것임
따라서 내가 원하는 특징을 추출해주는 기능은 없음
어떤 특징을 추출할지는 프로그래머가 결정해야함
이것이 머신러닝의 포인트


4. 회귀 분석
머신러닝에서는 회귀라는 단어를 많이 사용함 
회귀분석은 통계용어로 
Y가 연속된 값일때 Y = f(x) 와 같은 모델로 나타내는것

회귀에서 사용되는 가장 기본적인 모델은 Y = aX + b 이를 선형 회귀라고 부름
또한 이러한 모델에서 Y를 연속 측정의 종속변수(목표 변수),X를 독립 변수(설명 변수)fkrh qnfma
X가 1차원이라면 단순회귀 라고 부르며 2차원 이상이면 다중회귀라고 부름


5. 머신 러닝의 종류
5-1 교사 학습(Supervised learning) 지도학습

선생님이 옆에서 문제를 주고  답을 알려주는것과 같은 학습 방법임
데이터를 입력할떄 레이블(답) 을 함께 입력합니다. 
예 ) 글자 인식 

5-2 비교사 학습(unsupervised learning) 비지도 학습

최종적으로 내야하는 답이 정해져 있지 않음
일반적으로 사람도 제대로 알수 없는 본질적인 구조등을 확인할때 사용함

클러스터 분석(cluster) , 주성분 분석(principal component), 벡터 양자화(vector quantization) ,자기 조직화(Self organization)

5-3 강화학습(Reinforcement learning)

현재 상태를 관찰해서 어떻게 대응해야 할 지와 관련된 문제를 다룹니다
강화 학습은 교사학습과 비슷하지만 교사가  완전한 답을 제공하지 않는다는 것이 다름

강화학습에서는 행동의 주체와 환경(상황 또는 상태)이 등장함
행동의 주체는 환경을 관찰하고 ,이를 기반으로 의사결정을 내려 행동합니다.
이떄 환경이 변화하면서 행동의 주체가 어떤 보상을 받음
이를 기반으로 행동의 주체는 더 많은 보상을 얻을 수 있는 방향으로 행동을 학습하게 됨

예 ) 
행동의 주체 고양이 
환경은 자동으로 먹이주는 기계

고양이는 자동으로 사료 주는 기계에서 사료 냄새가 난다는 것을 발견함 
따라서 내부에 사료가 있다는 것은 알지만 어떻게 해야 사료를 먹을 수 있는지는 모름

그래서 고양이가 기계주변을 배회 하고 몸으로 비비고 손으로 쳐보기도 함
그렇게 여러 행동을 하다보니 스위치가 눌려 사료가 조금 나오고 고양이는 행동에 의해 사료라는 보상을 얻게 된것임

고양이는 어떤 상황에 사료가 나오는지 계속 반복해서 테스트 해보고 최종적으로 붉은색 버튼을 눌렀을떄 사료가 나온다는 것을 깨닫게도미

붉은색버튼을 누르면 먹이가 나오는것을 학습한것
이후에는 배가 고플때마다 버튼을 눌러 먹이를 먹을수 있을것임 고양이가 최선의 행동을 스스로 학습한것임
이러한것을 컴퓨터로 구현하는것이 강화 학습임

6. 머신러닝의 흐름

1.데이터 수집    머신러닝에서 데이터 수집은 제일 어려운 과정일수도 있음
2.데이터 가공   프로그램이 다루기 쉬운 형태로 데이터를 가공해야함
3.데이터 학습   
데이터 학습(3 - 1 학습방법 선택,3-2 매개변수 조정 3-3 모델 학습)
3-1 : 어떤 방법(알고리즘)을 사용해 학습을 시킬지 생각해야함 (SVM,랜덤포레스트,k-means)
svm 서포트 벡터 머신
2차원에서 머무르는것이 아닌 3차원등 차원을 높여서 데이터를 나누는 과정
고차원에서 데이터를 분류할수있는 칸막이 만드는것임
svm의 목표
Maximum margin hyperlane 을 찾는것 즉 마진을 최대화 하는 초평면을 찾는것

hyperlane 초평면
-데이터들을 분류할 수 있는 칸막이

support vectors 서포트 벡터
-초평면에서 가장 가까운 데이터

margin 마진
-양쪽 support vectors 와 hyperlane 간의 거리의 합

3-2 : 데이터와 알고리즘에 맞게 매개변수 지정
4. 모델 평가 
정밀도가 제대로 나오고 있는지
YES => 성공
NO = > 데이터 학습

7. 머신러닝의 응용분야

7-1 클래스 분류 
특정 데이터에 레이블을 붙여 분류할수 있음
예) 스팸 메일 분류 필기 인식, 증권 사기

7-2 클러스터링 그룹나누기
값의 유사성을 기반으로 데이터를 여러 그룹으로 나눌 수 있음 
예) 사용자의 취향을 그룹으로 묶어 사용자 취향에 맞는 광고를 제공하는 경우를 의미함

7-3 추천

특정 데이터를 기반으로 다른 데이터를 추천하는 것임
예를 들어 사용자가 인터넷 서점에서 구매한 책들을 기반으로 다른 책을 추천하는 경우를 의미함

7-4 회귀(Regression)
과거의 데이터를 기반으로 미래의 데이터를 예측하는것 
판매예측 주가 변동등을 예측하는 경우 

7-5 차원 축소 ( Dimensionality Reduction )
데이터의 특성을 유지하면서 데이터의 양을 줄이는것 
특성을 유지한 상태로 고차원의 데이터를 저차원의 데이터로 변환하는것
데이터를 시각화하거나 구조를 추출해서 용량을 줄여 계산을 빠르게 하거나 메모리를 절약할 때 사용

8.초과 학습(초과 적합)
초과학습(Overfitting) 이란 훈련 전용 데이터가 학습돼 있지만 학습되지 않은 새로운 데이터에 대해 제대로 된 예측을 못하는 상태를 의미함
배운것밖에 해결하지 못하는 상황을 의미함
이러한 상황이 일어나는 예) 데이터가 너무 적은 경우, 모델에 비해 문제가 너무 복잡한 경우
